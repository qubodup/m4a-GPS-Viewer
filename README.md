Map viewer for GPS data in filename and metadata of an m4a audio recording file. For GPS Audio Recorder ([Android](https://play.google.com/store/apps/details?id=com.gpsaudiorecorder), [iPhone/iPad](https://apps.apple.com/de/app/gps-audio-recorder/id6764623164)) output.

Example file: <a target="_blank" href="https://freesound.org/people/qubodup/sounds/856548/">LQ Bus Ride M4A with GPS Coordinates Metadata</a> by <a target="_blank" href="https://freesound.org/people/qubodup/">qubodup</a>
